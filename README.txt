Pong game README

How to play:

1. make program game.c on two UC funkits and face towards each other
2. The LED matrix will display "PONG!!"
3. Select which player you want to be by pushing the navigation switch towards or away from the led matrix
    one player must choose P1 whilst the other must choose P2.
4. When ready to play both players must push the nav switch down, player 1 wil start with the ball
5. Use the nav switch to move your paddle at the bottom of the screen in the way of the ball
6. hit the ball back and fourth until one player missies, the player that misses the ball first will lose
7. At the end of the game players will see a win or lose text on the matrix. To go back to the main menu press the navswitch down
